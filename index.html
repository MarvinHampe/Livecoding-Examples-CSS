Animations

Bounce Animation

Make an element bounce on the screen:

Example:


@keyframes bounce {
  0% { transform: translateY(0); }
  50% { transform: translateY(30px); }
  100% { transform: translateY(0); }
}

div.bounce {
  animation: bounce 2s infinite;
}
This is an example div with bounce animation.

Rotate Animation

Rotate an element continuously:

Example:


@keyframes rotate {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(-30deg); }
}

div.rotate {
  animation: rotate 3s linear infinite;
}
This is an example div with rotate animation.

Media Queries

List Media Queries

Change the list style based on screen size:

Example:


@media screen and (max-width: 700px) {
  .list-media-query {
    list-style-type: square;
  }
}

@media screen and (min-width: 701px) {
  .list-media-query {
    list-style-type: decimal;
  }
}
